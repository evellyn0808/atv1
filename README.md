Atividade 1-
Essa atividade é um exemplo básico de uma página HTML com JavaScript que exibe uma mensagem de boas-vindas personalizada.
O programa solicita o nome do usuário por meio de uma janela de diálogo e, em seguida, mostra uma saudação na própria página.

Atividade 2-
Essa atividade é uma página HTML com JavaScript que realiza a concatenação do nome e sobrenome do usuário a partir de um formulário e exibe uma mensagem de boas-vindas personalizada em uma janela de alerta.

Atividade 3-
Essa atividade consiste em um programa simples em JavaScript que realiza a soma de dois números inteiros informados pelo usuário.
Os valores são solicitados por meio de janelas de diálogo e o resultado da soma é exibido diretamente na página HTML.

Atividade 4-
Essa atividade é uma aplicação simples em HTML e JavaScript que simula uma máquina de preparo de sucos.
O usuário pode escolher uma ou mais frutas, selecionar o tipo de líquido, decidir se deseja açúcar e, ao final, receber uma mensagem personalizada com o suco montado.

Atividade 5-Essa atividade é uma calculadora simples em JavaScript que permite calcular o intervalo entre duas datas, exibindo a diferença em dias, meses e anos
